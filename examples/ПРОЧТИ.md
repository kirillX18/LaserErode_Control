# Примеры программ обработки (контуров)

Это примеры **программы обработки** для лазерно-эрозионной обработки —
плоского контура, по которому ведётся лазерная головка. Загружаются на
странице **«Процесс» → «Программа обработки» → «Выбрать…»**.

## Файлы

| Файл | Формат | Содержимое |
|------|--------|------------|
| `контур_звезда.dxf` | DXF (LWPOLYLINE + CIRCLE) | пятиконечная звезда в круглой рамке |
| `контур_звезда.svg` | SVG (polygon + circle)   | та же фигура |
| `контур_звезда.plt` | HPGL/PLT (PU/PD)         | контур звезды пером плоттера |

Все три описывают один и тот же контур — можно сравнить, как разные форматы
читаются одинаково.

## Поддерживаемые форматы

- **DXF** — LINE, LWPOLYLINE, POLYLINE/VERTEX, CIRCLE, ARC;
- **SVG** — line, polyline, polygon, rect, circle, ellipse, path (M/L/H/V/Z,
  кривые C/Q дробятся на отрезки);
- **PLT / HPGL** — перо PU (переход) и PD (рез).

## Что происходит при загрузке

1. Контур масштабируется под рабочее поле станка с сохранением пропорций и
   центрируется (X — ход головки, Y — ход стола).
2. Показывается в 3D на координатном столе (вкладка «Процесс»).
3. При запуске процесса **головка едет по контуру**: позиция вдоль траектории
   выбирается по доле прогресса (X задаёт головку, Y — стол), над точкой реза
   рисуется опускной луч.

3D-модель детали STL/OBJ по-прежнему можно загрузить — но она лишь
предпросмотр формы и траекторию не задаёт.
